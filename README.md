# Refactoring_MovieAPP

Скачайте ZIP гита и разахивируйте его!

Привет, чтобы запустить проект IOS нужно:

1) скачать в appstore на mac Xcode - https://apps.apple.com/us/app/xcode/id497799835?l=ru&mt=12
2) После запуска Xcode вы увидите иконку и нажмите "Open Existing Project
3) Выбирайте папку с проектом нажимайте, что доверяете разработчику
4) Далее, если у вас есть айфон, то можете в вверху посередине выбрать ваш iphone(в Iphone в дальнейшем подтвердите, что доверяете разработчику) и запустите приложение, приложение скачается на ваше Iphone и сразу запустится
5) Если у вас нет айфона, то выбирайте любую модель телефона из списка, подождите, пока загрузиться и у вас запуститься на MacBook Симулятор. 
6) Чтобы сделать связку с беком, введите ссылку API в ApiManager(он находиться в папке API и запустите:)


Удачи!
